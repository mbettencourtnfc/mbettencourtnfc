## What I'm working on

**[nordic-orphan-hta](https://github.com/mbettencourtnfc/nordic-orphan-hta)** — Exploring whether EMA orphan designation data can predict reimbursement outcomes across Nordic HTA bodies (TLV, Medicinrådet, Nye Metoder). Built a pipeline from web scraping to logistic regression, XGBoost, and SHAP. Main finding: public EMA metadata alone has essentially no predictive power (AUC ≈ 0.53) — the real drivers are ICER and clinical evidence, not captured at designation stage. Good learning on what data actually matters for these decisions.

*More coming: cost-effectiveness modelling, managed-entry agreement analysis.*
